This folder contains all utility scripts which can be used in project.

---------------------------------------------------------------------------------------------------------

ES_unit.R contains function ES_unit which can compute the expected short-fall for single asset/investment

VaR_unit.R contains function VaR_unit which can compute the value at risk for single asset/investment

VaR_unit_boot.R contains function VaR_unit_boot which bootstraps te upper bound of VaR

heatmap.R contains function heatmap. 

Fama_French_Research_daily.R is called to prepare data for Fama-French model.
